# Asha⁺

Asha⁺ is a modern, responsive PHC Operations Management System containing a React frontend and a NestJS backend.

## Project Structure

```
asha+/
├── frontend/  # React / Vite / Tailwind
├── backend/   # NestJS / Prisma / SQLite
├── README.md
└── .gitignore
```

## Setup Instructions

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run start:dev
```
